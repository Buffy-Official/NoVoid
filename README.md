# NoVoid Plugin
NoVoid will prevent players from falling into the void and teleports them back to the spawn point, useful for lobby with floating maps!

> [!WARNING]
> This plugin is still on Beta Test, please use this with caution. **Commits are unexpected and some code will change in the future.**

> [!NOTE]
> Want to try the plugin? Click [here](https://github.com/Buffy-Official/NoVoid/releases/latest) and place it into your plugins folder.
